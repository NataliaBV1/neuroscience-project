## project on neuroscience 

## Objective 
Investigate how asynchronous chaos arises in local neural circuits and what factors determine it, particularly in networks of excitatory and inhibitory (E-I) neurons.

## Project Content 
- Implementation of physical and mathematical models
- Numerical simulations
- Data analysis and visualization
- Generation of figures and graphs to interpret model behavior

## Methods and tools
- Python
- NumPy
- SciPy
- Matplotlib
- Jupyter Notebook
